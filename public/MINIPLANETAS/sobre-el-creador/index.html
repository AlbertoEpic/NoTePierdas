La mirada detrás de cada órbita

MINIPLANETAS nace de una forma de caminar el paisaje con calma: observar, esperar la luz justa y rescatar detalles que suelen pasar desapercibidos.

Cada escena se trabaja para conservar su atmósfera original y, al mismo tiempo, convertirse en un pequeño mundo que invita a detenerse, imaginar y volver a mirar.

Este proyecto une sensibilidad visual y técnica fotográfica para crear una colección donde cada mini-planeta tiene identidad propia.
